### 1️⃣ STEP 1: Persiapan Bahan
```bash
pkg update && pkg upgrade -y
pkg install nodejs git -y

2️⃣ STEP 2: Ambil Kode & Masuk Folder
git clone [https://github.com/kyy-dev/Wawa.git](https://github.com/kyy-dev/Wawa.git)
cd Wawa

3️⃣ STEP 3: Pasang Library (Wajib Cok)
npm install

4️⃣ STEP 4: Jalankan Bot & Scan QR
node index.js
